# Environment

My preferred environment configs for local development (shell config and theming).

- Install iTerm and import `iterm-profile.json`
- Install Homebrew
- Install ZShell with Oh My ZSH

```
brew install zsh
brew install autojump
brew install nvm
brew install git
```
